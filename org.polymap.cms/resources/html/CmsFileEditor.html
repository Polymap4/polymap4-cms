Hilfe zur Formatierung

Im Text kann sogenanntes Markdown verwendet werden. Das ist ein sehr einfache Art, um Formatierung (fett, kursiv, Überschriften, etc.) im Text zu markieren. Die komplette Anleitung zu Markdown enthält die Dokumentation zu allen Elementen. Nachfolgend findet sich eine kurze Übersicht.


So wird es geschrieben... ...und so sieht es dann aus
**fett** (oder __fett__) fett
*kursiv* (oder _kursiv_) kursiv
# Große Überschrift

## Diese ist etwas kleiner

### Und die ist noch kleiner

alternativ geht auch folgende Schreibweise:

Das ist eine große Überschrift
========================

Große Überschrift

Diese ist etwas kleiner

Und die ist noch kleiner
Ein Link zu einer normalen Seite:

[Anzeigetext](!Impressum.txt)

Ein Download-Link:

[Dokument](#media/dokument.pdf)

Ein Link zu einer externen Seite, wie Google

[Ein Link zu Google](http://google.de)

In eckigen Klammern wird der sichtbare Text des Links geschrieben. Danach folgt in runden Klammern der Name der Datei, auf die verwiesen wird. Angeführt von einem "!", wenn es ein Artikel ist und "#" bei einem Download.		 Anzeigetext
